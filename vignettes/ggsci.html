2.2 Sci-Fi Color Palettes

To fully reveal the potential of the package name ggsci, we also included several built-in color palettes inspired by sci-fi and comedy shows.

3 For ggplot2 graphics

As functions:

  • scale_color_npg()
  • scale_fill_npg()
  • scale_color_gradient_npg()

Remember to set aliases:

  • scale_colour_npg = scale_color_npg
  • scale_colour_gradient_npg() = scale_color_gradient_npg()

4 For non-ggplot2 graphics

To use in base graphics or lattice graphics, use ggsci_npg_discrete.

Color palettes also available as direct output functions, like in viridis.

5 Acknowledgements

Note some of the palettes might not be the best choice on aspects like color-blind safe, photocopy safe, or print friendly. If you have such considerations, you might want to check out the well-designed ColorBrewer palette and viridis palette.
